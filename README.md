# FitAI

A personal fitness trainer using GPT

## Streamlit app

1. Install requirements.txt: `pip install -r requirements.txt`

2. To run the app, `streamlit run ./app.py`

## FastAPI

1. Install requirements.txt: `pip install -r requirements.txt`
2. To run the Uvicorn server (API) on **port 5000**, `python ./main.py`
